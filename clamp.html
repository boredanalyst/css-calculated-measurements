Calculated Measurements

These are measurements that are products of mathematical operations.

clamp()

The browser evaluates first the expression in the clamp function. If it's between the provided min and max values, the expression will be used.

If the resulting value is either too high or too low, the browser will use the max or min values respectively.

                        The expression used for my height is 10vh + 20vh.
                        The minimum value is 10vh.
                        The maximum value is 20vh.
